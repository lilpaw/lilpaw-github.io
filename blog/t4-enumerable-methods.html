Enumerable Methods

Becoming aquainted with #map

Enumerable is a module - a group of methods packaged together that can "mixin" with other classes. This just means that the methods in the Enumerable module can be included with other classes, such as Array and Hash. As long as a class has its own #each method, it can use the Enumerable methods as well. The methods in the Enumerable module give useful ways to deal with elements in a collection.

I chose to the method #map for this blog post. It's one that I wasn't familiar. It's an alias for #collect, which I also wasn't familiar.

#map creates a new array from the results of running a block of code over the elements of a collection. #map! changes the original collection, rather than creating a new array, so be mindful of the exclamation point.

[5, 10, 20, 50].map {|x| x * 2}
#=> [10, 20, 40, 100]

pets = { "Charlie" => 6, "Mackie" => 9}
pets.map {|pet,age| age = age + 1}
#=> {Charlie" => 8, "Mackie" => 11}

Now, an example of #map!, which changes the original array.

names = ["brandi", "gabe", "charlie", "mackie"]
names.map! {|i| i.capitalize}

p names

#=> ["Brandi", "Gabe", "Charlie", "Mackie"]

This can be extremely useful when working with collections of objects. Any time you need to change or work with a collection, you can use #map to iterate through the elements and create a new collection as a result.
